# lib_matrix_opp.h
Library for operations with matrix

Function:
    1. Sum
    
    2. Sub
    
    3. Mul number
    
    4. Mul matrix on matrix
    
    5. Transpose
    
    6. Determinant
    
    7. Calculate complements
    
    8. Inverse matrix

Build library:
    
    make s21_matrix_oop.a

Run tests or create gcovr report:

    make tests
    make gcov_report

Run Debug in file test.cc:
    
    make run

Clean:
    
    make clean
